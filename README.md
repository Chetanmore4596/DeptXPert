# 📘 DeptXpert ERP

**DeptXpert** is a web-based ERP system developed as my **Diploma Final Year Project**.  
It is designed to help college departments manage **students, teachers, and departmental data** in a structured and efficient way.

---

## 🔹 Features
- 🔑 **Secure Login System** (Student, Teacher, HOD, Principal)
- 🧑‍🏫 **Teacher Management** – Add, update, and manage teacher data
- 🎓 **Student Management** – Maintain student records and results
- 🏫 **Department Dashboard** – Manage departmental details and faculty
- 📚 **Study Materials Upload** – Teachers can upload files/resources
- 🎮 **Quiz & Activities** – Interactive section for students
- 📊 **Principal Dashboard** – Overview of all departments

---

## 🔹 Tech Stack
- **Frontend**: HTML, CSS, JavaScript, Bootstrap  
- **Backend**: PHP  
- **Database**: MySQL (via XAMPP)

## 🔹 Installation & Setup
1. Install [XAMPP](https://www.apachefriends.org/index.html) (Apache + MySQL).  
2. Copy the project folder `DeptXpert` to `htdocs` directory inside XAMPP installation.  
   - Example: `C:\xampp\htdocs\DeptXpert`  
3. Start **Apache** and **MySQL** from XAMPP Control Panel.  
4. Import the SQL file:  
   - Open `phpMyAdmin` → Create new DB `deptxpert`  
   - Import `deptxpert.sql` from project folder  
5. Run the project in browser:  
   ```
   http://localhost/DeptXpert/
   ```
